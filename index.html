💡 Memory Tips & Shortcuts

📏 Length Tricks

  • 1 inch = 25.4mm - Think "quarter century"
  • 1m ≈ 1yd + 10% - 1m = 3.28ft ≈ 1.09yd
  • Foot = 0.3 - 1ft = 0.3048m (remember 0.3)
  • 12 inches = 1 foot - Always!

⚡ Force Shortcuts

  • kip = kilo-pound - 1 kip = 1000 lbf
  • 1kN ≈ 225lbf - Quarter kip
  • 4.45N per pound - 1lbf = 4.448N
  • Gravity: 9.81 m/s² or 32.2 ft/s²

💪 Stress Memory

  • 1MPa ≈ 145psi - Close to 150
  • Concrete 4ksi = 28MPa - Common strength
  • Steel 36ksi = 250MPa - Grade 36
  • PSI to kPa: ×7 - Actually 6.895

📐 Area & Volume

  • Square it! - Area = length² conversion
  • Cube it! - Volume = length³ conversion
  • 1m² ≈ 11ft² - Actually 10.76
  • 27 ft³ = 1 yd³ - 3×3×3

🏗️ Material Anchors

  • Concrete: 150 PCF = 2400 kg/m³
  • Steel: 490 PCF = 7850 kg/m³
  • Water: 62.4 PCF = 1000 kg/m³
  • Aluminum: 165 PCF = 2700 kg/m³

🔄 Moment Tricks

  • 12 inches rule - 1ft·lb = 12in·lb
  • 1kN·m ≈ 0.74 kip·ft - About 3/4
  • Multiply conversions - Force × Length
  • Right-hand rule - Thumb = axis

🧮 Mental Math

  • Round to 5 or 10 - For estimates
  • Use ratios - Know 1, multiply others
  • Check magnitude - MPa ~1/7 of ksi
  • Work backwards - Verify by reverse

⚠️ Avoid These!

  • PSI vs PSF - 144× difference!
  • kip vs kN - Different by 4.45
  • Mass vs Force - kg vs kgf
  • Ton vs Tonne - 2000lb vs 1000kg

🏭 Industrial Quick Ref

  • Pipe: NPS vs DN - Nominal sizes
  • Rebar: #4 = 1/2" - Bar number/8
  • Bolt: M12 = 1/2" - Metric/Imperial
  • HP = 0.746 kW - Power conversion

AISC Design Steps:
1. Vu = factored shear from beam
2. Choose bolt size & grade
3. Bolt shear capacity: φRn = φFnvAb
4. Bearing on tab: φRn = φ2.4FutD
5. Tab shear yield: φRn = φ0.6Fy×Ag
6. Tab shear rupture: φRn = φ0.6Fu×Anv
7. Check block shear: φRn = φ(0.6FuAnv+UbsFuAnt)

🔩 Moment Connections

End Plate Moment Connection

Flange Force: Tf = Mu / (d – tf)
Bolt tension: T/bolt = Tf / n_bolts
Check: T/bolt ≤ φFt×Ab
Plate thickness: tp = √(4Tf×g / φFy×bp×n)
Weld to beam: Full penetration groove

Flange Plate Moment Connection

Top/Bottom flange plates carry moment:
Plate force: F = Mu / d_beam
Plate area: Ap = F / (φFy)
Width: Same or wider than bf
Thickness: Ap / width
Bolts to column flange: F / φFnv×Ab

🔗 Splice Connections & Gusset Plates

Beam Splice

Web splice: takes shear + moment portion
Flange splice: takes flange force
Flange force: Ff = Mu/d × (Af/Ag)
Web moment: Mw = Mu × (Aw/Ag) × (d/2)
Splice plates: t ≥ 3/8", min 2 plates
Bolts each side: n = F / φRn per bolt

Column Splice (AISC D2)

Min splice strength: 50% col capacity
Butt plate or direct-weld to splice plates
Location: 4ft above floor level (pref)
For seismic: splice in middle 1/3 of column

Gusset Plate — Bracing Connection

Whitmore Width: lw = l×tan(30°) × 2 + w
where l = length of connection, w = bolt group width
Gusset axial: Pu = Whitmore × t × Fy
Min t gusset: P / (0.6Fy × lw)
Buckling length: 2/3 × L (Thornton method)
Seismic Gusset (AISC 341):
Linear clearance method: 2t offset from beam/col
Balanced Design Procedure (BDP)
Uniform Force Method (UFM) for forces

Block Shear — All Connections

φRn = φ[0.6Fu×Anv + Ubs×Fu×Ant]
≤ φ[0.6Fy×Agv + Ubs×Fu×Ant]
Ubs = 1.0 (uniform tension)
Ubs = 0.5 (non-uniform)
φ = 0.75

🏢 Column Base Plate Design

📐 Axial Load Base Plate — AISC Design Guide 1

STEP 1 — Required Bearing Area
A1_req = Pu / (φc × 0.85 × f'c)
φc = 0.60 (bearing), f'c = concrete strength
STEP 2 — Plate Dimensions
Try: B×N ≥ A1_req
N_min = d + 2(3") = d + 6 in (pref)
B_min = bf + 2(3") = bf + 6 in (pref)
Round up to nearest inch
STEP 3 — Actual Bearing Pressure
fp = Pu / (B×N) ≤ φ×0.85f'c×√(A2/A1)
A2 = frustum area at bearing depth
A2/A1 ≤ 4.0 (max multiplier)
STEP 4 — Plate Thickness
m = (N – 0.95d) / 2
n = (B – 0.80bf) / 2
n' = √(d×bf) / 4 (lambda factor)
l = max(m, n, λn')
t_req = l × √(2fp / (φFy))
φ = 0.90 for bending

📐 Moment Base Plate (P+M)

CASE 1 — Small Eccentricity (e ≤ N/6)
e = Mu/Pu
Full compression — no tension in bolts
fp,max = Pu/A + Mu×c/I
fp,min = Pu/A – Mu×c/I ≥ 0
Design plate for fp,max
CASE 2 — Large Eccentricity (e > N/6)
Anchor bolts in tension
Take moments about compression resultant:
T = Mu_new / d_bolts
C = Pu + T
fp = 2C / (3×B×Y)
Y = N/2 – e' (bearing length)
e' = N/2 – d_bolts/2
Plate thickness for Moment:
Cantilever length for tension side: l = d_bolt to plate edge
t = √(2Mu_plate / (φFy))
Mu_plate = T_bolt × l
